# ビデオスロット 仕様書

`/video_slot bet:<数値>` で起動する3リール・5ラインスロットマシン。

---

## 通常モード

### 基本フロー

1. `/video_slot bet:100` でスロット開始（bet分のポイント消費）
2. SPIN → 左リール自動停止(800ms) → 中リール自動停止(1000ms) → 右リール自動停止(1000ms)
3. ペッシリーチ時のみSTOPボタンで右リール手動停止

### リール構成

3リール × 3行。各リールは固定リールストリップ（21シンボル）を回転。

### 有効ライン（5ライン）

| ライン | 座標 |
|--------|------|
| 上段(横) | [0,0] [0,1] [0,2] |
| 中段(横) | [1,0] [1,1] [1,2] |
| 下段(横) | [2,0] [2,1] [2,2] |
| 斜め↘ | [0,0] [1,1] [2,2] |
| 斜め↗ | [2,0] [1,1] [0,2] |

| シンボル | 絵文字 | 配当（掛け金倍率） |

### ペッシ図柄（通常モード）

特殊シンボル。リール上には表示されず、内部抽選で決定。

#### ペッシ当選確率

| リール | 確率 |
|--------|------|
| 左リール | 1/16 |
| 左+中リール | 1/16 × 1/4 = 1/64 |
| 左+中+右リール | 1/16 × 1/4 × 1/2 = 1/128 |

#### ペッシ当選時の挙動

- ペッシ当選リールは強制的にペッシ図柄で表示
- ペッシが当選したリールはライン判定から除外（ペッシリーチ状態を含む）
- ペッシの支払い: ペッシリール数 × bet

#### 役抽選確率（通常モード）

| シンボル | 確率 |
|----------|------|
| GOD | 1/8192 |
| TheKuru | 1/170 |
| LEDX | 1/99 |
| ascendant1 | 1/64 |
| twitter | 1/40 |
| ornament | 1/20 |
| MarshmaoYummy | 1/9 |

### 通常モードの状態遷行

```
IDLE → SPINNING → LEFT_STOPPED → MID_STOPPED → FINISHED
                                    ↓ (左+中ペッシ時)
                                  PESSI_REACH → STOP → FINISHED
```

- ペッシリーチ（左+中がペッシ、右未決定）時のみSTOPボタン有効化
- ペッシリーチ以外は全自動停止

## ペッシボーナスモード

3リール全てペッシ（ペッシビット = 0b111）で通常モード終了時に突入。

### 基本フロー

1. 8ゲームのフリースピン
2. 各ゲーム: SPIN → STOPボタンで各リールを順次停止
3. 全リール停止後にゲーム結果表示 → 次ゲーム or ボーナス終了

### ボーナス中のペッシ抽選

各STOPボタン押下時、各シンボルスロット（行）ごとに独立してペッシ抽選:

| 結果 | 確率 |
|------|------|
| レインボーペッシ | 1/12 |
| 通常ペッシ | 1/5 |
| ハズレ | 残り |

- 既にconfirmedPessi（前ゲームまでに確定したペッシ）のリールは抽選スキップ
- 1シンボルスロットでもペッシが当選すればそのリール全体がペッシ確定
- レインボー当選時はペッシ確定＋レインボーフラグ（bit 7）付与

### ボーナス中の非ペッシリール

ペッシ非当選リールには役抽選を行わず、空白（⬛）を表示。

### ボーナス中の支払い

**ライン判定は行わず、最終的なペッシ停止リール数で計算:**

支払い = bet × ペッシリール数 × 2

| ペッシリール数 | 倍率 |
|---------------|------|
| 0 | x0 |
| 1 | x2 |
| 2 | x4 |
| 3 | x6 |
| N | x(N×2) |

8G完了時に一括計算・支払い。

### レインボーペッシバフ

レインボーペッシが出たゲームの結果時にランダムでバフ発動:

- **リール数+1**: 新しいリールを追加（上限7追加、合計10リール）
- **行数+1**: 新しい行を追加（上限7追加、合計10行）
- **ゲーム数+1**: 残りゲーム数+1

リール数・行数が上限(7追加)に達しているバフは候補から除外され、ゲーム数+1のみ選択される可能性がある。

### 拡張リール/行の挙動

- **リール数増加**: 新しいリールは `ReelStrips[reelIndex % 3]` を使用（リール0→3→0→3...の循環）
- **行数増加**: グリッドの行が増加、ペッシ連結表示も行数に応じて拡張
- **STOPボタン**: リール数に応じて動的に生成（3〜10リールに対応）
- **ペッシビットマスク**: bit 0〜6 で7リールまで対応（bit 7はレインボーフラグ）

### ペッシ連結表示

ペッシ図柄は常に縦に連結して表示。`long_pessi_1`（上）、`long_pessi_2`（中）、`long_pessi_3`（下）を使用:

| 行数 | 表示 |
|------|------|
| 1行 | <:pessi:> |
| 2行 | long_pessi_1 + long_pessi_3 |
| 3行 | long_pessi_1 + long_pessi_2 + long_pessi_3 |
| 4行以上 | long_pessi_1 + long_pessi_2 ×(行数-2) + long_pessi_3 |

通常モード・ボーナスモードともに、ペッシリールは常に縦連結表示。

ボーナス終了時の配当表示でも、確定ペッシリール数に応じて連結表示:

| ペッシ数 | 表示 |
|---------|------|
| 1リール | <:pessi:> |
| 2リール | long_pessi_1 + long_pessi_3 |
| 3リール以上 | long_pessi_1 + long_pessi_2 ×(N-2) + long_pessi_3 |

### ボーナスの状態遷行

```
BONUS_IDLE → BONUS_SPINNING → BONUS_STOPPING(stoppedReels=0) → STOP押下で順次停止
               (SPIN押下)                                    → 全リール停止 → BONUS_GAME_RESULT
                                              ↑                    |
                                              +------ STOP ------+
                                              | (次のリール待ち)  |
                                                                 (gamesLeft > 0)
                                                                   → SPIN → BONUS_SPINNING ...
                                                                 (gamesLeft <= 0)
                                                                   → SPIN → FINISHED (結果確認)
```

## Phase一覧

| 値 | Phase | 説明 |
|----|-------|------|
| 0 | IDLE | 初期状態・結果確認後 |
| 1 | SPINNING | 通常: リール回転中 |
| 2 | LEFT_STOPPED | 通常: 左リール停止 |
| 3 | MID_STOPPED | 通常: 中リール停止 |
| 4 | PESSI_REACH | 通常: ペッシリーチ（右リールSTOP待ち） |
| 5 | FINISHED | 通常: 全リール停止・結果表示 |
| 10 | BONUS_IDLE | ボーナス: SPIN待ち |
| 11 | BONUS_SPINNING | ボーナス: リール回転中 |
| 12 | BONUS_STOPPING | ボーナス: STOPボタン入力待ち（stoppedReelsで何リール停止済みか管理） |
| 14 | BONUS_GAME_RESULT | ボーナス: ゲーム結果表示 |
| 15 | BONUS_FINISHED | （未使用） |

## BonusData 構造

| フィールド | 型 | 説明 |
|-----------|------|------|
| bet | number | 掛け金 |
| gamesLeft | number | 残りゲーム数（初期値: 8） |
| confirmedPessi | number | 確定ペッシビットマスク（bit 0〜6: リール, bit 7: unused） |
| extraPessi | number | レインボーバフ等で付与された一時ペッシビット |
| positions | number[] | 各リールの停止位置 |
| currentPessi | number | 現ゲームのペッシ描画画報（bit 0〜6: リール, bit 7: レインボー） |
| rainbowThisGame | number | （未使用） |
| extraReels | number | 追加リール数（0〜7、合計3〜10リール） |
| extraRows | number | 追加行数（0〜7、合計3〜10行） |
| stoppedReels | number | 停止済みリール数（BONUS_STOPPING時使用） |

### ビットマスク

- **confirmedPessi**: bit 0 = リール0, bit 1 = リール1, ... bit 6 = リール6
- **currentPessi**: bit 0〜6 = ペッシ対象リール, bit 7 (RAINBOW_BIT = 128) = レインボーフラグ

---

## カスタムID形式

### 通常モード

```
vs-{action}-{userId}-{bet}-{phase}-{positions}-{pessiBits}
```

### ボーナスモード

```
vs-{action}-{userId}-{bet}-{phase}-{positions}-{pessiBits}-{gamesLeft}-{confirmedPessi}-{extraPessi}-{extraReels}-{extraRows}
```

- **action**: `spin`, `stop`, `bstop`
- **positions**: カンマ区切りの停止位置
- ボーナス状態は `bonusStates` Mapで保持（positionsとstoppedReelsはリストア時反映）
